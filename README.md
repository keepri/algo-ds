# Algorithms and Data Structures
`cargo run <usize> <bool>` arguments
- `usize` - custom sized Vec used for sort/search, default is 25000
- `bool` - disable logging, default is `true`

- bubble sort
- quick sort
- insertion sort

### Searching
- linear search
